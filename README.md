# TriagemIA Voice

Este é um projeto frontend desenvolvido com Vite e TypeScript.

## Pré-requisitos

*   **Node.js** (versão 16 ou superior)
*   **npm** ou **Yarn**

## Como Iniciar

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/FelipeSoaresAl/triagemia-voice.git
    ```
2.  **Acesse o diretório do projeto:**
    ```bash
    cd triagemia-voice
    ```
3.  **Instale as dependências:**
    ```bash
    npm install
    ```
    ou
    ```bash
    yarn
    ```
4.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
    ou
    ```bash
    yarn dev
    ```
5.  Abra seu navegador e acesse `http://localhost:5173`.

## Build

Para criar a versão de produção do projeto, execute:

```bash
npm run build
```
ou
```bash
yarn build
```
A versão de produção será criada no diretório `dist`.
